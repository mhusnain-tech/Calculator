# Calculator
Multi-functional C++ Calculator with a clean command-line interface.

Simple C++ Calculator:

A versatile command-line calculator built with C++ that handles basic arithmetic and specialized mathematical functions.

  **Key Features**
  
**User-Friendly Navigation:** Uses goto labels (change and repeat) to allow users to perform multiple calculations or change input numbers without restarting the program.

**Modular Design:** Every operation is organized into specific functions such as add(), subtract(), product(), and percentage() for better code structure.

**Specialized Math Tools:** Includes built-in support for calculating squares and square roots using the cmath library.

**Clean Visuals:** Features a custom starline() function to visually separate different operations and keep the console output organized.

**Percentage Calculator:** A dedicated mode to calculate percentages based on obtained and total marks.

  **How to Use**
  
Enter two initial numbers.

Choose an operation from the menu (1-10).

To exit, simply enter 0.
